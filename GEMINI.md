## 📝 Formato de Notas (Deducido de Clase 1)
Al generar o editar contenido en los archivos `.md`, se debe seguir este estilo:
- **Títulos:** Usar `##` para temas principales y `###`/`####` para subtemas.
- **Resaltado:** Usar `**negrita**` para conceptos clave y `*itálica*` para énfasis o ejemplos rápidos.
- **Código:**
  - Inline code (backticks) para variables y fragmentos cortos.
  - Bloques de código con sintaxis resaltada para ejemplos extensos.
- **Separadores:** Usar líneas horizontales (`---`) para dividir grandes secciones lógicas.
- **Estructura Didáctica:** Incluir siempre:
  - Definición del concepto.
  - Ejemplo práctico o fragmento de código.
  - Errores comunes (ej. `TypeError`).
